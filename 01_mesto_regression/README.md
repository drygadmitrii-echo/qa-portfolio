# Mesto — регресс по готовым тест-кейсам и баг-репорты

Этот кейс — про аккуратную базу: прогон по готовым тест-кейсам и фиксация дефектов так, чтобы их можно было воспроизвести без созвонов и уточнений.

## Что сделал
- Прогнал регресс по **8** готовым тест-кейсам
- Оформил **8** баг-репортов по результатам прогона
- Дополнительно собрал отдельную подборку из **15** багов по проекту

## Что лежит в папке
- test_cases_mesto_regression.xlsx — тест-кейсы + статусы прохождения
- bug_reports_mesto_regression.xlsx — баги по итогам прогона
- bug_reports_mesto_extra.xlsx — дополнительные баг-репорты

## Небольшое личное
Мне нравится, что тут всё по-честному: либо кейс проходит, либо нет — и сразу видно, насколько хорошо продукт держит базовые сценарии.
Особенно цепляют баги вокруг редактирования профиля — кажется мелочью, а по факту это одна из главных функций.

## Как открыть
Скачайте .xlsx и откройте в Excel или Google Sheets.
