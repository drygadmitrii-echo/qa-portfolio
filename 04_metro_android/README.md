# Яндекс.Метро (Android) — функциональные проверки, регресс и баги

Мобильный кейс: прогон функционального набора и регресса, фиксация дефектов с привязкой к окружению.

## Что сделано
- функциональный чек-лист: **36** проверок
- регрессионный чек-лист: **53** проверки
- баг-репорты: **25**

## Файл
- `metro_android_checklists_bugs.xlsx` — чек-листы + список дефектов

## Небольшое личное
В мобильных приложениях особенно “всплывают” нюансы: ориентация экрана, история действий, время/интервалы.
Мне понравилось, что здесь баги не из серии “что-то не так”, а вполне воспроизводимые и понятные — их очень четко видно.

## Как открыть
Скачайте `.xlsx` и откройте в Excel/Google Sheets.
